运行环境要求

  • Chrome 107+ 或 Edge 107+（WebCodecs 支持）
  • 通过 HTTP/HTTPS 服务器访问（WASM 限制）
  • 本地视频文件（H.264 / H.265 编码）
  • 无需安装任何依赖，全部通过 CDN 加载
